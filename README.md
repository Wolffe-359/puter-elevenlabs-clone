Puter Voice Studio

A browser-based text-to-speech application powered by Puter.js and ElevenLabs.

This project can be hosted entirely on GitHub Pages and does not require a backend server or API keys.

Features
ElevenLabs TTS through Puter.js
No API keys required
Runs entirely in the browser
Dark-mode interface
Text editor
Voice selection
Model selection
Audio format selection
Audio playback
MP3 download
Local generation history
Auto-save text
Batch generation mode
